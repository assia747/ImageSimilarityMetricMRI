------------------------ ReadMe --------------------

In order to run the function "ImageQualityAssessment.m", one has to add all folders and subfolders in the given
directory. This function takes a 2D image as an input and two optional parameters to be run. The output is the class 
label (an integer number between 1 and 5) for the given image. 

______________________________________________________

The functions and scripts for each chapter are located in the corresponding folders.

______________________________________________________

The simulation and modeling results can be found in the folder named "Results".
